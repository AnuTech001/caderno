Livro I

Durante 30 anos da chamada Era dos Computadores, o principal desafio desenvolver um hardware eficiente e de baixo custo. Contudo, atualmete é melhorar a qualidade, mas ainda com baixo custo das serviços de software.

As aplicações do software são:

  • Software básico
  • Software de tempo real
  • Software comercial, Web
  • Software científico ou de engenharia
  • Software embutido
  • Software de PC
  • Software de IA
  • Software de open source

Questionário I

Com base no questionário podemos afirmar que:

  • O software não se desgasta fisicamente pois ele não é físico como o hardware;
  • Que para o usuário focar nas características que agregam valor ao produto;
  • Na Engenharia de Software, o foco se baseia em técnicas e práticas utilizadas para desenvover programas;
  • Que a qualidade do programa depende do seu desenvolvimento;
  • Na Engenharia de Software também envolve ferramentas e métodos;
  • Que na maoiria das vezes aumentar a equipe quando há atraso no desenvolvimento do software pode gerar ainda mais atraso, pois exige trainarmento e adaptação dos novos membros;
  • Que cada software têm suas características específicas como:
    • Controle de equipamento = tempo real;
    • Simulação complexa = científico;
    • Sistemas empresariais = comercial;
    • Controle de dispositivos = embarcado.
  • Que mudanças de requisitos podem impactar o desenvolvimento do projeto;
  • Que os teste são cruciais para o sucesso do projeto;
  • Que sistema baseados em software são fundamentais para o desenvolvimento de aplicações modernas.
